# 🎣 Core Fishing Loop

# Версия: 0.1
# Статус: draft

## 🧱 Состояния

idle → waiting → bite → reeling → reset

---

### idle
игрок ещё не закинул

---

### waiting
- поплавок в воде
- ожидание

---

### bite
- окно реакции (1–2 сек)
- поплавок дёргается

---

### reeling
- поплавка нет
- идёт вываживание
- работает натяжение

---

## 🎮 Управление

### Мышь
- клик → заброс / подсечка
- зажал → тянешь
- отпустил → ослабил

### Клавиатура
- Space → то же

---

## 🔁 Поток логики

1. Игрок кликает → handleCast()
2. delay → bite
3. реакция:
   - вовремя → reeling
   - нет → reset
4. в reeling:
   - удержание → рост натяжения
   - отпуск → падение
5. ошибки:
   - слишком сильно → обрыв
   - слишком слабо → сход

# 🧱 Архитектура проекта

# Версия: 0.1
# Статус: draft

## 📁 Структура проекта

src/
├─ app/
├─ screens/
├─ components/
├─ services/
├─ utils/
├─ styles/

---

## 📄 Назначение слоёв

### components/
Переиспользуемые UI-компоненты  
НЕ содержат логики

---

### screens/
Контейнеры экранов  
Содержат:
- состояние
- управление фазами

---

### services/game/
Игровая логика:
- расчёты
- механики

---

### data/
Данные:
- рыбы
- снасти
- локации

---

## ❗ Архитектурные правила

- игровая логика НЕ в React
- UI только отображает
- расчёты → services/game
- data = только данные
